# EventHub Frontend

Tela principal em React para descobrir e gerenciar eventos, conectada aos endpoints do backend Spring Boot.

## Como rodar

Vá para 
https://github.com/DevOps-EventHub/eventhub-infra

```bash
npm install
npm run dev
```

O Vite sobe em `http://localhost:5173` e redireciona chamadas `/api` para `http://localhost:8080`.
